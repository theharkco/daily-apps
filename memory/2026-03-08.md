## MoodCanvas 🎨
**Stack**: SvelteKit, TypeScript, Tailwind CSS v4

Created **MoodCanvas** - a collaborative mood-based art generator that transforms emotions into visual abstract artwork. Different from recent Next.js/Python days, went with SvelteKit for its reactive magic!

### Features
- Generate abstract artwork based on mood (joy, melancholy, excitement, calm)
- Color theory-based palettes for each emotion
- Beautiful dark theme UI with smooth animations
- Gallery view with mood filtering
- LocalStorage persistence for your creations
